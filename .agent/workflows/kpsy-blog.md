# KPSY LAB 블로그 작성 마스터 가이드

KPSY LAB의 블로그는 단순한 정보의 나열이 아니라, 독자의 고정관념을 해체하고 새로운 **'사유의 질서'**를 세워주는 공간입니다.

---

## 1. 핵심 철학: 글의 기둥 (Core Pillars)

모든 포스팅은 작성 전 아래 세 가지 질문을 통과해야 합니다.

| 원칙 | 질문 |
|------|------|
| **맥락적 존재** | 이 현상을 '좋다/나쁘다' 혹은 '유형'으로 규정하지 않고, 어떤 '맥락'에서 발현되는지로 풀어내고 있는가? |
| **질서와 깊이** | 독자가 30초 만에 훑어보는 '요약'이 아니라, 5분간 몰입하여 사유하게 만드는 '깊이'가 있는가? |
| **비이분법적 통찰** | 상충하는 가치(예: 이기심과 생존, 불안과 각성) 사이의 회색지대를 정교하게 다루고 있는가? |

---

## 2. 블로그 아키텍처 (Standard Structure)

모든 글은 **3단 레이어 구조**를 가집니다. 각 섹션은 **최소 1,000자 이상**의 밀도를 지향합니다.

### Layer 1: 해체 (The Deconstruction)
> 우리가 상식이라고 믿는 것들에 균열을 냅니다.

- **목표**: 독자의 방어 기제를 무너뜨리고 호기심을 자극함
- **방법**: 당연하게 여겨지는 심리적 통념을 비판적으로 분석
- **예시**: "열정은 사실 뇌의 보상 회로가 타버리기 직전의 위험한 신호일 수 있습니다."

### Layer 2: 기제 (Neural & Psychological Mechanism)
> 현상의 이면에 숨겨진 과학적 질서를 드러냅니다.

- **목표**: 논리적 엄밀함과 지적 권위 확보
- **방법**: 뇌과학(편도체, 도파민 경로, 전두엽 등)과 심리학 이론을 동원하여 현상을 분해

### Layer 3: 재구성 (Re-contextualization)
> KPSY LAB만의 공식을 통해 대안을 제시합니다.

- **목표**: 독자가 자신의 삶에 적용할 수 있는 '도면'을 제공
- **방법**: 핵심 공식 적용

```
Potential = Performance - Interference
```

무엇이 **'방해 요인(I)'**인지 규명하고 이를 **'성능(P)'**으로 전환하는 법을 제안합니다.

## 3. 문체 및 서술 가이드 (Style & Grammar)

| 구분 | ❌ 지양할 것 | ✅ 지향할 것 |
|------|-------------|-------------|
| **톤** | 친절하고 따뜻한 위로 | 차가운 지성과 날카로운 통찰 |
| **어휘** | 데이터, 정보, 연결, 결과 | 사유의 궤적, 인지적 질서, 맥락적 배치 |
| **결론** | "모두 행복해지세요" 식의 마무리 | 새로운 질문을 던지는 열린 결말 |
| **분량** | 800~1,200자의 가벼운 분량 | **3,000자 이상**의 고밀도 장문 |

### 추가 규칙
- **반복 금지**: "말씀하신 대로", "요약하자면"과 같은 불필요한 연결어는 지능을 낮아 보이게 하니 모두 삭제할 것
- **구체적 수치 및 명칭**: 막연한 '심리학 연구' 대신 실제 이론가(예: 칼 융, 다마지오 등)나 신경학적 명칭을 정확히 사용할 것
- **수식의 활용**: 글 중간에 LaTeX 형식을 활용하여 논리적 가시성을 높일 것

## 4. 실전 체크리스트 (Final Audit)

작성 완료 후 반드시 점검:

- [ ] 글 전체에서 '낙인'이나 '정의'를 내리는 표현이 있는가? → 있다면 **'맥락'**으로 수정
- [ ] `P = p - i` 공식이 글의 핵심 로직으로 녹아들어 있는가?
- [ ] 50번 이후 포스트처럼 단조로운 패턴이 반복되지는 않는가?
- [ ] 독자가 이 글을 읽고 나서 "나를 좀 더 이해하게 됐다"가 아니라 **"내 마음의 구조를 보게 됐다"**고 느낄 것인가?

## 사용 예시

블로그 포스트 작성 요청 시:
```
/kpsy-blog 주제: [주제명]
```

이 가이드의 모든 원칙을 적용하여 3단 구조로 3,000자 이상의 고밀도 포스트를 작성합니다.
