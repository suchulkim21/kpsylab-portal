## 시작하기

### KPSY LAB이란?

KPSY LAB은 심리 분석 서비스를 제공하는 통합 플랫폼입니다. 다양한 심리 측정 도구와 분석 서비스를 통해 자신을 더 잘 이해할 수 있도록 도와드립니다.

### 제공 서비스

- **MNPS**: Dark Tetrad 심리 분석
- **Second Genesis**: 심층 행동 패턴 분석
- **블로그**: 심리학 관련 전문 콘텐츠
- **게시판**: 커뮤니티 소통 공간

## 회원가입 및 로그인

### 회원가입

1. 메인 페이지에서 **"회원가입"** 버튼 클릭
2. 다음 정보를 입력하세요:
   - **사용자명**: 3-20자 (필수)
   - **이메일**: 유효한 이메일 주소 (필수)
   - **비밀번호**: 최소 6자 이상 (필수)
3. **"가입하기"** 버튼 클릭

#### 회원가입 규칙
- 사용자명: 3-20자 사이
- 이메일: 유효한 이메일 형식이어야 함
- 비밀번호: 최소 6자 이상
- 이메일과 사용자명은 중복 불가

### 로그인

1. 메인 페이지에서 **"로그인"** 버튼 클릭
2. 다음 중 하나를 입력:
   - **이메일** 또는 **사용자명**
   - **비밀번호**
3. **"로그인"** 버튼 클릭

#### 로그인 팁
- 이메일 또는 사용자명 모두 사용 가능
- 비밀번호는 대소문자 구분
- 로그인 상태는 7일간 유지됩니다

### 로그아웃

1. 우측 상단 사용자 메뉴 클릭
2. **"로그아웃"** 선택

## 주요 서비스

### 1. MNPS (Dark Tetrad 분석)

**MNPS**는 마키아벨리즘, 나르시시즘, 사이코패시, 사디즘을 측정하는 심리 분석 서비스입니다.

#### 사용 방법

1. 상단 메뉴에서 **"MNPS"** 클릭
2. 서비스 소개를 확인한 후 **"테스트 시작"** 클릭
3. 질문에 솔직하게 답변
4. 분석 결과 확인

#### 결과 해석

- 각 특성별 점수 제공
- 상세한 분석 리포트
- 개인화된 인사이트

### 2. Second Genesis

**Second Genesis**는 행동 패턴과 의사결정 메커니즘을 심층 분석하는 서비스입니다.

#### 모듈 구성

**Module 1: 간섭 분석**
- 내면의 간섭 요소 파악
- 억압된 패턴 분석

**Module 2: 행동 패턴 분석**
- 3단계 시나리오 테스트
- Proactivity, Adaptability, Social Distance 측정
- 추가 질문으로 정확도 향상

**Module 3: 에너지 분석**
- 에너지, 집중력, 창의성 분석

#### 사용 방법

1. 상단 메뉴에서 **"Second Genesis"** 클릭
2. 원하는 모듈 선택
3. 각 단계별 질문에 답변
4. 중간 결과 확인 후 계속 진행
5. 최종 리포트 확인

#### 신뢰도 향상

- Module 2에서 신뢰도가 낮을 경우
- **"추가 질문 진행하기"** 버튼 클릭
- 추가 질문에 답변하여 정확도 향상

## 블로그 활용

### 블로그 둘러보기

1. 상단 메뉴에서 **"블로그"** 클릭
2. 최신 포스트 목록 확인
3. 관심 있는 포스트 클릭하여 상세 내용 확인

### 블로그 검색

1. 블로그 페이지 상단 검색창 사용
2. 키워드 입력 후 검색
3. 제목 또는 내용에 키워드가 포함된 포스트 확인

### 주요 콘텐츠

- Dark Tetrad 관련 심리학 콘텐츠
- 행동 패턴 분석 기사
- 심리학 연구 결과
- 자기계발 관련 글

## 게시판 사용

### 게시판 접근

1. 상단 메뉴에서 **"게시판"** 클릭
2. 게시글 목록 확인

### 게시글 작성

1. **"글쓰기"** 버튼 클릭
2. 제목과 내용 입력
3. **"작성하기"** 클릭

**주의**: 게시판 이용 시 서비스 약관을 준수해주세요.

### 게시글 조회

1. 게시판에서 관심 있는 게시글 클릭
2. 상세 내용 확인
3. 댓글 확인 및 작성 (기능 구현 시)

## 문의하기

### 문의 방법

1. 상단 메뉴에서 **"문의"** 클릭
2. 문의 양식 작성:

## FAQ

### 계정 관련

**Q: 비밀번호를 잊어버렸어요.**  
A: 현재 비밀번호 재설정 기능이 준비 중입니다. 문의하기를 통해 관리자에게 연락해주세요.

**Q: 이메일을 변경할 수 있나요?**  
A: 현재 이메일 변경 기능은 제공되지 않습니다. 새 계정을 생성하거나 문의하기를 통해 관리자에게 연락해주세요.

**Q: 계정을 삭제할 수 있나요?**  
A: 문의하기를 통해 계정 삭제 요청을 보내주시면 처리해드립니다.

### 서비스 관련

**Q: 테스트 결과는 어디에 저장되나요?**  
A: 테스트 결과는 브라우저의 로컬 스토리지에 저장됩니다. 브라우저 캐시를 삭제하면 결과가 삭제될 수 있습니다.

**Q: 테스트 결과를 다시 볼 수 있나요?**  
A: Second Genesis의 경우, 테스트 완료 후 결과 페이지에서 다시 확인할 수 있습니다.

**Q: 여러 번 테스트할 수 있나요?**  
A: 네, 언제든지 다시 테스트할 수 있습니다. "다시 진단하기" 버튼을 사용하세요.

**Q: 테스트 결과의 정확도는 어떤가요?**  
A: 테스트는 과학적으로 검증된 도구를 기반으로 하지만, 참고용입니다. 전문적인 심리 상담을 원하시면 전문가와 상담하시기 바랍니다.

### 기술 관련

**Q: 모바일에서도 사용할 수 있나요?**  
A: 네, 반응형 디자인으로 모바일, 태블릿, PC 모두에서 사용 가능합니다.

**Q: 어떤 브라우저를 지원하나요?**  
A: Chrome, Firefox, Safari, Edge 등 최신 브라우저를 지원합니다.

**Q: 페이지가 로드되지 않아요.**  
A: 브라우저 캐시를 삭제하거나 새로고침해보세요. 문제가 계속되면 문의하기를 통해 알려주세요.
