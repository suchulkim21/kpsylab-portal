# 블로그 포스트 확장 작업 리스트

## 작업 개요
20개 블로그 포스트의 내용을 각 3천자 이상으로 확장하는 작업

## 현재 상태
- ✅ 이미지 URL 수정 완료 (w=800 → w=1200)
- ✅ 유사도 테스트 완료 (모두 20% 이하)
- ❌ 포스트 내용 확장 미완료

## 통계
- 현재 포스트 수: 20개
- 현재 평균 길이: 233자
- 목표 길이: 3,000자 이상
- 필요한 확장량: 약 13배 증가 (각 포스트 약 2,767자 추가 필요)

## 작업 항목

### 1. 포스트 내용 확장 (각 포스트를 3천자 이상으로)
다음 20개 포스트를 각각 3천자 이상의 상세한 내용으로 확장해야 합니다:

1. **MNPS 서문: 어둠의 4요소(Dark Tetrad)란 무엇인가?** (현재: 424자)
   - 어둠의 4요소에 대한 심층 설명
   - 각 요소의 역사적 배경
   - 실생활 사례 및 적용
   - 연구 동향 및 최신 지견

2. **현대 사회의 마키아벨리즘: 직장 내 정치의 기술** (현재: 284자)
   - 마키아벨리즘의 정의 및 특징
   - 직장 내에서의 구체적 사례
   - 대응 방법 및 방어 전략
   - 조직 차원의 해결책

3. **나르시시즘의 두 얼굴: 거대함과 취약함** (현재: 282자)
   - 거대성 나르시시즘과 취약성 나르시시즘의 상세 비교
   - 각 유형의 행동 패턴
   - 관계에서의 영향
   - 치료 및 개선 방법

4. **사이코패스와 소시오패스: 냉혈한의 차이점** (현재: 241자)
   - 선천적 vs 후천적 특성
   - 뇌 구조 및 생물학적 차이
   - 범죄학적 관점
   - 진단 및 식별 방법

5. **디지털 사디즘: 인터넷 트롤링의 심리학** (현재: 233자)
   - 온라인 사디즘의 특성
   - 익명성의 영향
   - 피해자 관점
   - 사이버 범죄 및 법적 대응

6. **가스라이팅: 현실을 조작하는 마키아벨리적 전술** (현재: 211자)
   - 가스라이팅의 다양한 기법
   - 단계별 패턴 분석
   - 피해 회복 방법
   - 전문가 상담 및 치료

7. **성공한 CEO들에게서 보이는 '어둠의 3요소'** (현재: 212자)
   - 리더십과 어둠의 성격
   - 성공과 비윤리의 딜레마
   - 조직 문화에 미치는 영향
   - 바람직한 리더십 모델

8. **공감 능력의 결여: 차가운 공감(Cold Empathy)** (현재: 252자)
   - 인지적 공감 vs 정서적 공감
   - 차가운 공감의 작용 메커니즘
   - 대인관계에서의 영향
   - 공감 능력 향상 방법

9. **연애 권력 게임: 나르시시스트의 사랑 방식** (현재: 249자)
   - 이상화-평가절하-폐기 사이클
   - 나르시시스트 공급(Narcissistic Supply)
   - 피해자의 심리 상태
   - 탈출 및 회복 전략

10. **도덕적 허가(Moral Licensing): 착한 행동 뒤의 위선** (현재: 204자)
    - 도덕적 허가의 심리 메커니즘
    - 일상생활에서의 사례
    - 자기기만과 위선
    - 진정한 도덕성 확립

11. **SNS 시대의 보여주기식 삶과 MNPS** (현재: 190자)
    - 소셜 미디어와 나르시시즘
    - 인상 관리와 허상
    - 정신 건강에 미치는 영향
    - 디지털 웰빙 실천법

12. **조용한 위험: 수동 공격성(Passive Aggression)** (현재: 193자)
    - 수동 공격성의 다양한 형태
    - 조직 내 갈등의 원인
    - 대응 방법
    - 건강한 소통 방법

13. **진화심리학으로 본 어둠의 성격** (현재: 237자)
    - 진화적 적응 전략
    - 빠른 인생 전략(Fast Life History Strategy)
    - 짝짓기 전략과 성격
    - 현대 사회에서의 의미

14. **유머 감각과 지능, 그리고 사디즘** (현재: 202자)
    - 블랙 코미디의 심리학
    - 공격적 유머와 성격
    - 지능과 유머의 관계
    - 건강한 유머 감각

15. **협상의 기술: 마키아벨리스트를 이기는 법** (현재: 212자)
    - 마키아벨리스트의 협상 전술
    - 감정 조작 대응법
    - Win-Win 구조 만들기
    - 실전 협상 전략

16. **그림자 통합(Integrate the Shadow): 칼 융의 해법** (현재: 219자)
    - 융의 분석심리학과 그림자
    - 그림자 통합 과정
    - 개성화(Individuation)의 여정
    - 실천 방법 및 예시

17. **사랑에 빠진 사이코패스? 불가능한 신화인가** (현재: 196자)
    - 사이코패스의 관계 패턴
    - 소유욕과 사랑의 혼동
    - 질투와 집착의 심리
    - 안전한 탈출 방법

18. **MNPS 성향이 예술과 창작에 미치는 영향** (현재: 197자)
    - 예술가들의 성격 특성
    - 고통과 창의성
    - 천재성의 어두운 면
    - 건강한 창작 활동

19. **아이들에게서 나타나는 냉담함(Callous-Unemotional)** (현재: 209자)
    - CU 특성의 조기 발견
    - 아동기 개입의 중요성
    - 효과적인 교육 방법
    - 전문가 도움 받기

20. **진정한 리더십: 어둠을 넘어선 빛** (현재: 213자)
    - 서번트 리더십
    - 진정성 리더십
    - 조직 문화 개선
    - 지속 가능한 성장

## 작업 방법

### 스크립트 사용
포스트 확장 작업을 위해 `apps/portal/scripts/expand-blog-posts.js` 스크립트를 생성하여:
1. 데이터베이스에서 현재 포스트 읽기
2. 각 포스트 내용을 3천자 이상으로 확장
3. 유사도 테스트 재실행 (20% 이하 확인)
4. 데이터베이스 업데이트

### 참고 파일
- `apps/portal/scripts/test-similarity.js` - 유사도 테스트 스크립트
- `apps/portal/scripts/fix-blog-images.js` - 이미지 URL 수정 스크립트
- `mnps/_LEGACY/populate_blog.js` - 원본 포스트 데이터

## 완료 조건
- [ ] 모든 포스트가 3천자 이상
- [ ] 유사도 테스트 통과 (모든 쌍이 20% 이하)
- [ ] 이미지 URL 확인 (w=1200)
- [ ] 브라우저에서 최종 확인

## 예상 소요 시간
각 포스트를 전문적이고 상세하게 작성하는 작업이므로:
- 포스트당 약 30-60분 소요 예상
- 총 20개 포스트: 약 10-20시간 소요 예상

## 주의사항
1. 각 포스트의 주제와 일관성 유지
2. 심리학적으로 정확한 정보 제공
3. 실용적이고 이해하기 쉬운 설명
4. 유사도가 20%를 초과하지 않도록 내용 차별화
5. HTML 태그를 포함한 구조화된 형식 유지
